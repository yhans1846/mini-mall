# Mini Mall 🛍️

微型电商平台，基于 Next.js 16 全栈开发。包含前台商城与后台管理系统。

> 本项目完全通过 AI 编程实现：**DeepSeek** 生成核心代码 → **Claude Code** 审查优化 → 双模型协作迭代完成全部功能，零手写代码。

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | Next.js 16 (App Router) |
| 语言 | TypeScript |
| 数据库 | SQLite (Prisma ORM) |
| 样式 | TailwindCSS 4 |
| 认证 | next-auth v5 (Credentials + JWT) |
| 加密 | bcryptjs |
| 校验 | zod |
| 客户端请求 | SWR |
| 通知 | sonner |
| 图表 | recharts（后台仪表盘） |

## 功能特性

### 前台商城

- 首页：轮播 Banner / 分类入口 / 限时秒杀 / 热销排行 / 新品上市 / 品牌故事
- 商品浏览：搜索、分类筛选、排序、分页
- 商品详情：图片轮播、品牌、规格参数、标签、产地、重量、视频
- 购物车：数量修改、删除、会员折扣预估
- 结算下单：地址管理、会员折扣、库存扣减
- 订单管理：状态筛选、模拟支付
- 会员中心：等级展示、累计消费、升级进度
- 心悦会员体系：3 级折扣（9.8折 / 9.5折 / 9折）

### 后台管理

- 仪表盘：统计卡片、Recharts 图表（收入/月趋势/状态分布）、热销商品、库存预警
- 商品管理：CRUD、批量上架/下架/删除
- 订单管理：状态筛选、状态流转（PAID→SHIPPED→COMPLETED）
- 分类管理、秒杀活动管理、用户管理
- 数据统计：销售统计、商品统计、用户统计
- CSV 导出

## 快速开始

### 环境要求

- Node.js >= 18
- npm

### 安装

```bash
git clone <repo-url>
cd mini-mall

# 安装依赖
npm install

# 同步数据库
npm run db:push

# 导入种子数据（含测试用户、秒杀商品）
npm run db:seed

# 修改.env.example 文件名称为 .env 并修改AUTH_SECRET的值
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000
